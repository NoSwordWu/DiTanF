# 浙江省市场拜访分析驾驶舱

一个基于 Web 的浙江省市场拜访数据分析平台，用于统计和分析浙江省各地市、区县的公共机构拜访登记数据。

## 🌟 功能特性

### 核心功能
- **热力图展示**：按地市/区县维度展示各机构类型的拜访覆盖情况
- **数据统计**：实时统计总拜访次数和本周拜访次数
- **趋势分析**：近七日拜访趋势图表
- **地市分布**：各城市拜访数据柱状图分析
- **机构分布**：各类型机构拜访数据饼图分析

### 交互功能
- **拜访登记**：双击热力图单元格可打开拜访登记弹窗
- **历史记录管理**：支持查看和删除历史拜访记录
- **日期筛选**：支持按日期范围筛选数据
- **主题切换**：支持深色/浅色主题切换

## 🔑 关键技术要点

### 数据持久化方案

#### 生产环境（Netlify）
项目部署到 Netlify 时，采用 **Netlify Functions + Netlify Blobs** 实现数据持久化：

1. **API 端点**：`/.netlify/functions/visits`
   - GET 请求：从 Blobs 读取数据
   - POST 请求：保存数据到 Blobs

2. **降级策略**：当 API 不可用时（如本地开发），自动降级到浏览器 `localStorage`

3. **文件位置**：
   - 服务端函数：[netlify/functions/visits.mjs](file:///e:/TraeProgram/DiTanF/netlify/functions/visits.mjs)
   - 前端调用：[index.html 中的 saveData 函数](file:///e:/TraeProgram/DiTanF/index.html#L956-L968)

#### 本地开发环境
- 首次加载时从 [data/marketData.json](file:///e:/TraeProgram/DiTanF/data/marketData.json) 读取种子数据
- 数据变更保存到 `localStorage`，刷新页面后数据不丢失

### 图表绑定机制

所有图表通过 ECharts 初始化，支持实时更新：

| 图表 | 函数 | 更新时机 |
|------|------|----------|
| 近七日趋势 | `updateSevenDayChart()` | 新增/删除拜访记录时 |
| 地市覆盖分析 | `updateCityChart()` | 筛选条件变化时 |
| 机构覆盖分析 | `updateOrgChart()` | 筛选条件变化时 |

### 热力图渲染

热力图采用动态颜色计算，根据拜访次数自动调整颜色深浅：

```javascript
// 颜色渐变：蓝色 → 绿色
intensity = Math.min(count / 12, 1);
// RGB 计算：蓝色(50,100,200) → 绿色(50,230,150)
```

## 🚀 部署指南

### 部署到 Netlify（推荐）

1. **推送到 GitHub**
   ```bash
   git add .
   git commit -m "Initial commit"
   git push origin main
   ```

2. **连接 Netlify**
   - 登录 [Netlify](https://app.netlify.com/)
   - 点击 "Add new site" → "Import an existing project"
   - 选择 GitHub 仓库，Netlify 自动识别配置

3. **验证部署**
   - 访问 `https://<your-site>.netlify.app/.netlify/functions/visits`
   - 应返回 JSON 数据

### 本地开发

直接在浏览器中打开 `index.html` 即可运行，数据自动保存到 `localStorage`。

## 📊 数据统计说明

### 统计指标
- **总拜访次数**：所有记录的累计拜访次数（支持日期筛选）
- **本周拜访次数**：本周一至今日的拜访次数

### 图表维度
- **近七日趋势**：按日期统计拜访次数
- **地市覆盖**：按城市统计拜访次数（支持总计/本周切换）
- **机构覆盖**：按机构类型统计拜访次数（支持总计/本周切换）

## 📝 使用说明

1. **查看数据**：打开首页即可看到热力图和统计数据
2. **登记拜访**：双击热力图单元格打开拜访登记弹窗
3. **筛选数据**：使用顶部日期选择器筛选特定时间段数据
4. **展开/折叠地市**：点击地市行可展开查看下属区县数据
